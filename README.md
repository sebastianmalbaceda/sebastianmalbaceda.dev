# sebastianmalbaceda.dev

Personal website built with [Astro](https://astro.build) + TypeScript + Tailwind CSS.

## Quick start

```bash
npm install
npm run dev      # localhost:4321
npm run build    # dist/
```

## Deploy

Static site — just point Cloudflare Pages to this repo:

- **Build command:** `npm run build`
- **Output directory:** `dist/`
